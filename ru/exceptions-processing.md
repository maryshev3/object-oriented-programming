# Обработка ошибок

----
План:
1) Демонстрации примера реализации функции деления на 0. Если пользователь ввёл 0 в качестве делителя, то это явно ошибка, которую стоит обрабатывать.
2) Стоит ввести понятие исключения - объект, содержащий сведения о возникших проблемах. Стоит модифицировать функцию так, чтобы делалась проверка на 0 и генерировалось исключение (throw).
3) Надо рассказать о встроенных типах исключений: о том, что приоритетно надо использовать именно их.
4) Надо показать, как обрабатывать исключения, т.е. рассказать про try catch.
5) Сказать про то, что если встроенных типов исключений не хватает, то можно создать свой унаследовавшись от exception.
----
